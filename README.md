# 🥭 Mango Detection and Counting System using YOLOv5

## 📌 Overview
This project is a web-based mango detection and counting system built using the YOLOv5 deep learning model. It automates fruit yield estimation by detecting mangoes in images and real-time video streams.

The system replaces manual counting methods with an AI-powered solution that improves accuracy, efficiency, and scalability in agricultural operations.

---

## 🚀 Features
- 🔍 Real-time mango detection using YOLOv5
- 🖼️ Image upload detection
- 🎥 Live video stream detection
- 📊 Automatic mango counting
- 📦 Bounding box visualization
- 🌐 User-friendly web interface

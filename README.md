# Webproject
MANGO DETECTION AND COUNTING SYSTEM
Mango Detection and Counting System using YOLOv5

This project presents a web-based intelligent system for detecting and counting mangoes using the YOLOv5 deep learning model. The system is designed to automate fruit yield estimation by identifying mangoes in both static images and real-time video streams.

Traditional manual counting methods in agriculture are time-consuming, labor-intensive, and prone to errors. To overcome these challenges, this project leverages computer vision and deep learning techniques to provide an accurate, efficient, and scalable solution for precision agriculture.

The model is trained on a custom dataset of over 4,000 annotated mango images captured under diverse environmental conditions, ensuring robustness in real-world scenarios. It achieves a high detection accuracy with a mean Average Precision (mAP@0.5) of approximately 93%, making it reliable for practical use.

The system is deployed as a web application using Flask for backend processing and OpenCV for image handling. Users can upload images or use a live camera feed to detect mangoes, visualize bounding boxes, and obtain the total count instantly.

🚀 Key Features
Real-time mango detection using YOLOv5
Image upload and live video stream support
Accurate fruit counting with bounding box visualization
User-friendly web interface
High detection accuracy (~93% mAP)
Scalable architecture for multi-fruit detection
